# ocean_claim

## Yêu cầu
- Taì khoản phải claim ít nhất 1 lần 

## Chuẩn bị môi trường
- Nodejs v18
- Chạy lệnh  `npm i`
- Thêm seed phrase vào wallets.txt

## Chạy script
- `node index.js`


# Gửi Ocean

- Thay địa chỉ ví muốn chuyển ocean đến ở recipient trong file sendOcean.js
- `node sendOcean.js`
